Theory
Introduction to Chatbot

A chatbot is an Artificial Intelligence application that interacts with users through text or voice communication.

Chatbots are widely used in:

customer support,
banking,
e-commerce,
healthcare,
and online services.
Smart Customer Support Chatbot

This chatbot simulates an online store assistant.

It can handle:

greetings,
order tracking,
refunds,
delivery information,
product inquiries.

The chatbot identifies user intention using:

keyword matching,
pattern recognition.
Intent Recognition

Intent recognition means identifying:

what the user wants.

Example:

User Input	Intent
"track my order"	order_status
"refund my item"	refund
"delivery time"	delivery
Regular Expressions

The chatbot uses:

re.search()

to extract order IDs from user messages.

Applications
Customer support systems
Virtual assistants

FAQ automation

24×7 availability
Faster response
Reduced human workload
Improved customer experience
Algorithm
1. Start chatbot.
2. Display greeting message.
3. Accept user input.
4. Detect user intent.
5. Process corresponding response.
6. Display chatbot reply.
7. Repeat until user exits.
