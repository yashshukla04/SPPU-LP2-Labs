Title

Installation and Configuration of Google App Engine

Aim

To install and configure Google App Engine.

Objectives
To understand Google App Engine.
To learn cloud-based web hosting.
To understand PaaS cloud model.
Theory

Google App Engine is a Platform as a Service (PaaS) cloud platform provided by Google.

It allows developers to:

build,
deploy,
and scale web applications automatically.

Features:

Automatic scaling
High security
Reliability
Platform independence
Cost savings

Applications deployed on App Engine can support millions of users.

Best Buy
Khan Academy
