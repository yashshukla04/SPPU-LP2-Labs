Title

Setup of SaaS Cloud using VMware ESXi over LAN

Aim

To setup SaaS cloud over LAN using VMware ESXi cloud platform.

Objectives
To configure private cloud.
To understand SaaS cloud infrastructure.
To configure VMware ESXi environment.
Theory

VMware ESXi is a hypervisor used for virtualization and cloud infrastructure setup.

It allows:

creation of virtual machines,
resource management,
clustering,
virtualization.

Important Concepts:

Datacenter
Cluster
Host
Virtual Machines
Network configuration

ESXi supports:

virtualization,
cloud deployment,
resource sharing.
