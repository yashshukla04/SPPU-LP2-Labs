Theory

Amazon EC2 (Elastic Compute Cloud) is a web service provided by Amazon Web Services (AWS) that allows users to create virtual servers in the cloud.

An EC2 instance is a virtual machine that can run applications on AWS infrastructure.

Features of EC2:

On-demand virtual servers
Scalable computing
Pay-as-you-use pricing
Multiple operating system support
Secure cloud infrastructure

Important Terms:

AMI (Amazon Machine Image): Template containing operating system.
Instance Type: Hardware configuration of VM.
VPC: Virtual Private Cloud.
Subnet: Division of VPC network.
Security Group: Virtual firewall.

AWS allows users to launch instances based on CPU, RAM, and storage requirements.
