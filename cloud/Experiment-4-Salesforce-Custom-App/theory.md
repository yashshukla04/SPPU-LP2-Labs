Title

Design and Development of Custom Application using Salesforce Cloud

Aim

To design and develop a custom application using Salesforce Cloud.

Objectives
To learn Salesforce cloud administration.
To create custom cloud applications.
Theory

Salesforce is a cloud-based CRM platform.

Salesforce provides:

Sales Cloud
Service Cloud
Community Cloud
Lightning Platform

Lightning Platform allows developers to build cloud applications using:

Apex
Visualforce
Lightning Components

Custom applications help organizations automate business processes.
